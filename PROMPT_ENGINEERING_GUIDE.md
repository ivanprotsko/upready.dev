## 1. ФУНДАМЕНТАЛЬНЫЕ ПРИНЦИПЫ

### 1.1 Главное правило: думай как кинематографист, а не как зритель

Разница между любительским и профессиональным промптом -- это разница между **"опиши что видишь"** и **"дай техническое задание съёмочной группе"**.

**Любитель:** *"Beautiful woman in a coffee shop"*
**Профессионал:** *"Candid portrait of a woman in her 30s at a corner cafe, caught mid-laugh, shot on Leica M11, 50mm f/1.4, Rembrandt lighting from window, Kodak Portra 400, visible grain in shadows"*

Профессиональный промпт -- это ТЗ для 5 специалистов одновременно:
- **Оператор (DP):** камера, объектив, диафрагма, экспозиция
- **Художник по свету:** схема света, направление, температура, модификаторы
- **Арт-директор:** композиция, цветовая палитра, мудборд
- **Художник по костюмам/декорациям:** материалы, текстуры, реквизит
- **Колорист:** цветокоррекция, плёночный сток, пост-обработка

**Каждый элемент, который ты НЕ указал -- модель решает за тебя. Профессионал контролирует то, что важно, и оставляет свободу там, где можно.**

### 1.2 Физика вместо прилагательных

**Стоп-слова (никогда не используй):**
- "beautiful", "stunning", "amazing", "gorgeous"
- "ultra-realistic", "hyper-realistic", "8K" (без контекста)
- "masterpiece", "best quality" (для SD-моделей допустимо как буст)
- "highly detailed" (лучше описать КАКИЕ детали)

**Замены:**
| Вместо этого | Используй это |
|---|---|
| "beautiful lighting" | "Rembrandt lighting with warm 3200K key light from upper-left" |
| "realistic skin" | "visible skin pores, subsurface scattering, natural asymmetry" |
| "blurred background" | "85mm f/1.4, shallow depth of field, creamy circular bokeh" |
| "cinematic" | "shot on ARRI Alexa, anamorphic, 2.39:1, teal-orange grade" |
| "high quality" | "shot on Phase One IQ4 150MP, f/8, studio strobe" |

### 1.3 Правило 3-5 компонентов

Промпт не должен содержать больше **3-5 ключевых компонентов**. Конфликтующие инструкции производят хуже, чем недосказанность.

**Конфликты (избегай):**
- "Photorealistic cartoon" -- взаимоисключающие концепции
- "Minimalist design with intricate details" -- модель пытается удовлетворить оба, получается ни то ни другое
- "Bright sunny day with moody dramatic shadows" -- физическое противоречие

## 2. АРХИТЕКТУРА ПРОМПТА

### 2.1 Универсальная формула

```
[Субъект + действие] → [Среда/контекст] → [Стиль/медиум] → [Техника (камера, свет)] → [Настроение/атмосфера] → [Исключения]
```

### 2.2 Четыре профессиональных подхода

#### A. Подход "Оператор-постановщик"
Промпт описывает физику съёмки: камеру, объектив, плёнку, свет.

```
Extreme macro close-up of a weathered craftsman's hands shaping clay,
shot on Sony A7R IV with 100mm macro lens, f/2.8, single overhead
softbox with warm gel, visible clay particles in air catching light,
subsurface scattering on wet clay surface, Kodak Portra 800
```

#### B. Подход "Арт-директор"
Промпт -- это бриф для команды: что, где, зачем, какое впечатление.

```
Hero image for premium craft pottery brand. Close-up of artisan's
weathered hands on spinning wheel. Must convey: heritage, authenticity,
human touch. Warm earth-tone palette. Generous negative space on right
for text overlay. Editorial photography, luxury magazine quality.
```

#### C. Подход "Кинематографист"
Фокус на оптических явлениях, световых схемах, цветокоррекции.

```
Film noir scene, 10:1 lighting ratio, single practical street lamp
creating venetian blind shadow patterns on wet brick, cigarette smoke
catching volumetric light beam, shot on vintage Cooke Panchro lenses,
Kodak Double-X B&W stock, deep blacks, blown highlights, 2.39:1
anamorphic aspect ratio
```

#### D. Подход "Разделение стиля и контента"
Создаёшь отдельный Style Prompt и Content Prompt, потом комбинируешь.

**Style Prompt (переиспользуемый):**
```
Shot on Hasselblad medium format, 100mm f/2.2, shallow depth of field,
Kodak Portra 400 color science, subtle film grain, warm directional
window light, muted earth-tone palette, editorial photography
```

**Content Prompt (уникальный):**
```
Portrait of a 60-year-old craftsman with weathered hands holding
handmade ceramic bowl, deep wrinkles, salt-and-pepper beard
```

## 3. ПОРЯДОК СЛОВ И ПРИОРИТИЗАЦИЯ

### 3.1 Правило: первые слова -- самые важные

Во ВСЕХ моделях слова в начале промпта получают больший вес. Это архитектурная особенность текстовых энкодеров.

**CLIP (SD/SDXL):** Эффективная длина ~20 токенов. После 75 токенов -- разбивка на чанки, каждый обрабатывается отдельно.

**T5 (Flux):** До 512 токенов, но начало всё равно приоритетнее.

**GPT Image (gpt-image-1/1.5):** До 32,000 символов, обрабатывает как естественный язык. Менее чувствителен к порядку, но front-loading всё равно рекомендуется.

### 3.2 Рекомендуемый порядок по модели

| Приоритет | Stable Diffusion | Midjourney | GPT Image | Flux |
|---|---|---|---|---|
| 1 | Субъект | Субъект | Нарративная сцена | Субъект |
| 2 | Медиум/качество | Стиль/настроение | Эмоциональный контекст | Действие/состояние |
| 3 | Стиль | Техника | Визуальные детали | Стиль |
| 4 | Освещение | Композиция | Стилевая референция | Контекст/свет |
| 5 | Композиция | --no (исключения) | Техника (опционально) | Камера/техника |
| 6 | Детали/текстура | --ar (формат) | | Настроение/палитра |

## 4. МОДЕЛИ И ИХ ОСОБЕННОСТИ

### 4.1 GPT Image (gpt-image-1, gpt-image-1.5)

**Архитектура:** Нативная мультимодальная авторегрессия (текст + изображение в одной модели).
**Лимит промпта:** 32,000 символов.
**Размеры:** 1024x1024, 1536x1024, 1024x1536, auto.
**Качество:** low, medium (default), high, auto.

**Ключевые особенности:**
- НЕ переписывает промпт (в отличие от DALL-E 3)
- Понимает промпты как естественный язык, а не ключевые слова
- НЕ поддерживает числовые веса (word:1.3)
- Негативные промпты -- через естественный язык: "No watermark, no text, avoid oversaturation"
- Для приоритизации: поставь важное первым, опиши подробнее, используй "The most important aspect is..."
- gpt-image-1.5 на ~4x быстрее, лучше текст, меньше артефактов

**Оптимальная структура:**
```
[Сцена/фон] → [Субъект] → [Ключевые детали] → [Техника/камера] → [Ограничения]
```

### 4.2 Midjourney (V7)

**Лимит промпта:** ~60 слов эффективно.
**Ключевые параметры:**

| Параметр | Диапазон | Назначение |
|---|---|---|
| --ar | Любое (16:9, 3:4, 1:1) | Соотношение сторон |
| --s (stylize) | 0-1000 (default 100) | Художественная свобода. 0-50: строго по промпту. 500+: MJ берёт творческую свободу |
| --c (chaos) | 0-100 (default 0) | Разнообразие между 4 вариантами |
| --w (weird) | 0-3000 (default 0) | Степень "странности" |
| --no | ключевые слова | Исключение элементов |
| --sref URL/CODE | URL или число | Стилевая референция |
| --cref URL | URL | Референция персонажа |
| --oref URL | URL (V7) | Omni-референция (высшая точность) |
| --seed | 0-4294967295 | Воспроизводимость |

**Формула профессионального промпта:**
```
[Субъект + действие] [среда + атмосфера] [стиль/медиум] [освещение] [камера/объектив] [цветовая палитра] --ar X:Y --s N --sref CODE
```

**Про --sref:** Это ключевой инструмент стилевой консистентности. Можно:
- Использовать URL изображения
- Числовой код из библиотеки (midlibrary.io, srefcodes.com)
- `--sref random` для исследования
- Смешивать: `--sref CODE1 CODE2`
- Контролировать силу: `--sw 10-1000`

### 4.3 Stable Diffusion (SDXL) / Flux

**SDXL:**
- Двойной CLIP энкодер (768-d + 1280-d = 2048-d)
- 77 токенов на чанк
- Поддержка весов: `(word:1.3)`, `[word]` (ослабление)
- Негативные промпты -- полная поддержка
- CFG Scale: 5-8

**Flux:**
- Двойной энкодер: CLIP-L (ключевые слова) + T5-XXL (естественный язык)
- **КРИТИЧЕСКОЕ ОТКРЫТИЕ:** Энкодерам нужны РАЗНЫЕ промпты!
  - T5: полные описательные предложения
  - CLIP-L: ключевые слова через запятую
  - Идентичные промпты обоим = деградация качества на 50-75%
- НЕТ нативных негативных промптов
- CFG Scale: 3.0-3.5 (фотореализм), 3.5-6.0 (стилизация)

## 5. ВИЗУАЛЬНЫЙ СЛОВАРЬ: ОБЯЗАТЕЛЬНЫЕ ПАРАМЕТРЫ

Эти параметры ДОЛЖНЫ быть указаны в каждом профессиональном промпте. Пропуск любого из них -- передача контроля модели.

### 5.1 ОСВЕЩЕНИЕ (наивысший impact-per-word)

Одно слово об освещении меняет результат больше, чем пять прилагательных.

#### Направление и качество света

| Термин | Эффект |
|---|---|
| `Rembrandt lighting` | Треугольник света на щеке, классический портрет, глубина и драма |
| `rim light / edge light` | Светящийся контур вокруг субъекта, отделяет от фона |
| `butterfly lighting` | Тень-бабочка под носом, гламурный/модный свет |
| `split lighting` | Половина лица в свете, половина в тени, драматичный/резкий |
| `loop lighting` | Маленькая тень от носа сбоку, универсальный портретный свет |
| `three-point lighting` | Ключевой + заполняющий + контровой, стандартная студия |
| `volumetric lighting / god rays` | Видимые лучи через туман/пыль, кинематографично, духовно |
| `chiaroscuro` | Экстремальный контраст свет/тень (Караваджо), скульптурное качество |
| `high-key lighting` | Яркое, равномерное, минимум теней, коммерческий/позитивный |
| `low-key lighting` | Доминирующие тени, выборочное освещение, нуар/драма |

#### Цветовая температура

| Термин | Визуал |
|---|---|
| `warm tungsten 2700K` | Оранжево-тёплый, уютный, домашний |
| `daylight balanced 5500K` | Нейтральный белый, чистый |
| `cool blue 7500K` | Холодный, клинический, загадочный |
| `mixed color temperature` | Тёплые + холодные источники, визуальное напряжение |

#### Время суток

| Термин | Визуал |
|---|---|
| `golden hour` | Тёплый янтарно-золотой горизонтальный свет, длинные тени |
| `blue hour` | Глубокий синий с остаточным свечением горизонта |
| `harsh midday sun` | Жёсткие тени сверху, высокий контраст |
| `overcast diffused light` | Равномерный, мягкий, без теней |
| `candlelight` | Тёплый, мерцающий, интимный |

### 5.2 КОМПОЗИЦИЯ

| Термин | Эффект |
|---|---|
| `rule of thirds` | Субъект на пересечении третей, не по центру |
| `golden ratio / Fibonacci spiral` | Элементы по спирали 1:1.618 |
| `negative space` | Пустота вокруг субъекта, изоляция, элегантность |
| `leading lines` | Линии ведут взгляд к субъекту |
| `frame within a frame` | Арки, двери, окна обрамляют субъект |
| `depth layering` | Передний план, средний план, фон = объём |
| `Dutch angle` | Наклонённая камера, тревога/напряжение |

#### Типы кадров

| Термин | Что показывает |
|---|---|
| `extreme close-up` | Деталь (глаз, губы, текстура) |
| `close-up` | Голова и плечи, эмоции |
| `medium shot` | От пояса, разговорный |
| `full body shot` | Целиком, поза и одежда |
| `wide / establishing shot` | Вся среда, контекст и масштаб |

#### Ракурсы

| Термин | Эффект |
|---|---|
| `eye level` | Нейтральный, relatable |
| `low angle looking up` | Мощность, героизм, доминирование |
| `high angle looking down` | Уязвимость, подчинение |
| `bird's eye view` | Паттерны, масштаб, раскладка |
| `worm's eye view` | Грандиозность, монументальность |
| `over-the-shoulder` | Связь между персонажами, глубина |

### 5.3 КАМЕРА И ОБЪЕКТИВ

#### Фокусное расстояние

| Объектив | Визуальный эффект |
|---|---|
| `14mm fisheye` | Экстремальное искажение, 180 градусов, сюрреализм |
| `24mm wide angle` | Широкая перспектива, искажение на краях, архитектура |
| `35mm` | Естественный взгляд, близко к человеческому зрению, документальный |
| `50mm` | Самый близкий к глазу, натуральные пропорции, классика |

## 6. ВИЗУАЛЬНЫЙ СЛОВАРЬ: ПАРАМЕТРЫ ДЛЯ КРЕАТИВНОСТИ

Эти параметры можно варьировать для достижения уникального стиля.

### 6.1 ПОСТ-ОБРАБОТКА И ОПТИЧЕСКИЕ ЭФФЕКТЫ

| Термин | Эффект |
|---|---|
| `film grain` | Аналоговая текстура, тепло, аутентичность |
| `halation` | Красно-оранжевое свечение вокруг ярких источников (CineStill) |
| `chromatic aberration` | Цветная бахрома по краям, реализм, несовершенство оптики |
| `lens flare` | Блики от линзы, кинематографично |
| `anamorphic lens flare` | Горизонтальные синие полосы, big-budget кино |
| `vignette` | Затемнение по краям, фокус на центр |
| `light leak` | Тёплые цветные полосы, аналоговая случайность, ностальгия |
| `bokeh balls` | Размытые круги от огней на фоне, мечтательность |
| `bloom / soft glow` | Светлые области расплываются, эфирность |
| `tilt-shift` | Выборочный фокус, эффект миниатюры |

### 6.2 ТЕКСТУРЫ И МАТЕРИАЛЫ

| Материал | Как описать |
|---|---|
| Бетон | `raw concrete, brutalist, visible aggregate, form marks` |
| Старое дерево | `weathered wood, silver-gray patina, visible grain splitting` |
| Ржавый металл | `oxidized steel, orange-brown patina, flaking layers, pitted texture` |
| Мрамор | `Carrara marble, veined, cool translucent, polished` |
| Кожа | `full-grain leather, aged, visible wear pattern, rich browns` |
| Бархат | `crushed velvet, deep color absorption, directional nap` |
| Шёлк | `flowing silk, lustrous sheen, light catches on folds` |
| Матовый пластик | `matte surface, no reflections, even diffused appearance` |
| Хром | `polished chrome, perfect reflections, cool blue tint` |
| Стекло | `hand-blown glass, refraction, caustic light patterns` |

### 6.3 НАСТРОЕНИЕ И АТМОСФЕРА

| Настроение | Что сигнализирует модели |
|---|---|

## 7. ТИПИЧНЫЕ ОШИБКИ И АНТИПАТТЕРНЫ

### 7.1 Почему изображения выглядят как стоковые фото

| Ошибка | Исправление |
|---|---|
| Общие прилагательные ("beautiful", "stunning") | Конкретные технические термины ("rim-lit", "desaturated teal palette") |
| Нет указания света | Хотя бы одна схема (Rembrandt, rim light, volumetric) |
| Дефолтный формат (1:1) | Осознанный формат (3:4 портрет, 16:9 кино, 2.39:1 анаморф) |
| Не указаны камера/объектив | `shot on [camera], [lens]` |
| Слово "photorealistic" без контекста | Конкретное оборудование + плёнка + схема света |
| Идеально чистая кожа | `skin texture, pores, imperfections, natural asymmetry` |
| Субъект по центру | Композиция: `rule of thirds`, `off-center`, `negative space` |
| Нет атмосферы | `atmospheric haze`, `dust particles`, `morning mist` |

### 7.2 "AI-шный" вид: что его вызывает

**Корневые причины:**
1. **Over-smoothing:** пластиковая кожа и поверхности
2. **Идеальная симметрия:** реальные лица асимметричны
3. **Неестественная резкость:** всё в идеальном фокусе
4. **Дефолтная цветокоррекция:** teal-and-orange или перенасыщенность
5. **Слишком идеальный свет:** равномерный, льстящий, без несовершенств

**Как убрать AI-look:**
1. Добавь плёночное зерно: `subtle film grain, Kodak Portra 400`
2. Добавь несовершенства: `slight asymmetry, stray hairs, micro-scratches`
3. Укажи конкретную камеру вместо "realistic"
4. Опиши кандидный момент: `caught mid-stride, genuine expression`
5. Понизь stylize в Midjourney: `--s 50-150`

### 7.3 Потеря деталей

**Причина:** Attention decay -- детали в конце длинного промпта получают меньше внимания.

**Решения:**
- Front-load критичные детали в первые 20 токенов
- Используй синтаксис весов: `(critical detail:1.3)`
- Группируй связанные слова в один семантический блок
- Для Flux: используй JSON-структуру для сложных сцен

## 8. МЕТОДОЛОГИЯ ИТЕРАЦИИ

### 8.1 Три прохода

**Проход 1 -- Исследование:**
Широкое описание, высокий chaos (в MJ). Цель: увидеть интерпретацию модели.
```
Fantasy forest scene --c 50 --s 100 --ar 16:9
```

**Проход 2 -- Направление:**
Добавь специфику на основе результатов Прохода 1. Зафиксируй seed удачного варианта.
```
Enchanted forest with bioluminescent mushrooms, moonlit clearing,
shot on ARRI Alexa, 35mm lens --c 20 --s 150 --seed 12345
```

**Проход 3 -- Финал:**
Полная техническая спецификация, плёнка, точная схема света, материалы.
```
[Полный промпт с 5 компонентами] --c 5 --s 200 --seed 12345 --sref CODE
```

### 8.2 A/B тестирование

**Золотое правило: меняй только ОДНУ переменную за раз.**

1. Фиксируй baseline промпт + seed
2. Вариант A: меняешь освещение
3. Вариант B: меняешь объектив
4. Сравниваешь с baseline
5. Внедряешь лучшее в новый baseline
6. Повторяешь

### 8.3 Управление seed'ами

- Один seed + один промпт = воспроизводимый результат
- Зафиксировав seed, можно изолировать влияние каждого изменения в промпте
- Веди таблицу: seed | промпт | оценка | заметки

## 9. СТИЛЕВОЙ ТРАНСФЕР ЧЕРЕЗ ТЕКСТ

### 9.1 Метод "Style Prompt + Content Prompt"

1. Попроси LLM проанализировать работу конкретного художника/фотографа
2. Извлеки стилевые характеристики в отдельный Style Prompt
3. Комбинируй Style Prompt с разным Content Prompt для консистентности

### 9.2 Описание техники вместо имени автора

Вместо "in the style of [Artist]" -- опиши технику:
- **Мазки:** `visible impasto technique, thick palette knife application`
- **Цвет:** `complementary warm-dominant scheme, amber-to-teal gradient`
- **Композиция:** `strong diagonal leading lines, golden ratio spiral`
- **Медиум:** `oil-on-canvas, transparent shadow glazes, visible canvas weave`

### 9.3 Совмещение стилей

**Правило:** Совмещай стили, которые пересекаются хотя бы в одном измерении (цвет, свет, композиция), но различаются в других.

**Хорошие комбинации:**
- Film noir + cyberpunk (схожий свет, разная эстетика)
- Реалистичная фото + акварельные штрихи (грунт + текстура)

**Плохие комбинации:**
- Гиперреализм + мультяшная стилизация (полный конфликт)
- Импрессионизм + фотореализм (взаимо-отмена)

## 10. ТЕСТОВАЯ МЕТОДОЛОГИЯ

### 10.1 Метрики оценки

#### Метрика 1: Ожидания пользователя
- Прочти описание задачи (бриф/контекст блока)
- Сгенерируй изображение
- Оцени 1-10: "Соответствует ли результат тому, что ожидал увидеть?"
- Учитывай: эмоциональный отклик, уместность для контекста, wow-фактор

#### Метрика 2: Стилистическое соответствие референсам
- Предоставь 2-3 референсных изображения стиля
- Сгенерируй изображение с промптом, описывающим этот стиль
- Оцени 1-10 по параметрам:
  - Цветовая палитра (совпадение тональности)
  - Характер освещения (направление, жёсткость, температура)
  - Текстурность (зерно, мазки, гладкость)
  - Общее "ощущение" стиля

#### Метрика 3: Соответствие деталей промпта
- Составь чек-лист всех деталей из промпта
- Для каждой детали: присутствует/частично/отсутствует
- Процент совпадения = (присутствует / всего) * 100
- Приемлемо: > 80%. Профессионально: > 90%

#### Метрика 4: Стиль через текст vs через референс
- **Тест A:** Промпт с подробным стилевым описанием БЕЗ референса
- **Тест B:** Тот же промпт + референсное изображение стиля (через --sref / img2img)
- Сравни A и B по Метрике 2
- Цель: разница < 2 баллов = текстовое описание стиля достаточно

### 10.2 Протокол тестирования

1. **Выбери 3-5 стилистически уникальных референсов** (неизвестные художники/иллюстраторы)
2. **Для каждого:**
   a. Проанализируй стиль: палитра, свет, текстура, композиция, настроение
   b. Сформулируй Style Prompt (только текст, без показа изображения модели)
   c. Сгенерируй 3-5 изображений
   d. Оцени по всем 4 метрикам
3. **Документируй:** промпт, seed, параметры, оценки
4. **Итерируй:** улучшай Style Prompt на основе провалов
5. **Финальный тест:** сгенерируй 10 изображений с финальным промптом, оцени среднее

## 11. СПРАВОЧНЫЕ ТАБЛИЦЫ

### Быстрая шпаргалка: Структура промпта по модели

#### GPT Image (gpt-image-1.5)
```
[Сцена описательным предложением]. [Субъект с деталями].
[Техника: камера, объектив, свет]. [Стиль и настроение].
No [исключения].
```

#### Midjourney V7
```
[субъект], [среда], [стиль], [свет], [камера], [палитра]
--ar 3:4 --s 120 --sref CODE --no text watermark
```

#### Stable Diffusion / SDXL
```
(masterpiece:1.2), (photorealistic:1.4), [субъект], [медиум],
[стиль], [свет], [камера], (деталь:1.1), [плёнка]

Negative: cartoon, anime, illustration, blurry, deformed,
bad anatomy, plastic skin, oversaturated
```

#### Flux (двойной промпт)
```
T5: "Полное описательное предложение о сцене, субъекте,
действии, атмосфере и стиле."

CLIP-L: "ключевые, слова, через, запятую, камера, свет, стиль"
```

### Совместимость параметров модели

| Возможность | GPT Image | Midjourney | SDXL | Flux |
|---|---|---|---|---|
| Негативные промпты | Через текст | --no | Полные | Нет |
| Веса слов | Нет | :: | (word:N) | Нет |
| Seed | Нет | --seed | Полный | Полный |
| Стилевой референс | Нет | --sref | LoRA/IP-Adapter | LoRA/IP-Adapter |
| Прозрачный фон | Да | Нет | Нет | Нет |
| Рендер текста | Отличный | Хороший (V7) | Слабый | Хороший |
| Макс. промпт | 32000 chars | ~60 слов | 75 токенов/чанк | 512 токенов T5 |
