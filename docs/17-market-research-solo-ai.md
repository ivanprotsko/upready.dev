## 5. Сегменты клиентов

### 5.1 Non-technical founders (крупнейший сегмент)

- **Бюджет:** $3K-15K
- **Профиль:** Бизнес/маркетинг бэкграунд, первый стартап
- **Боли:** Страх перед техникой, miscommunication с разработчиками, feature bloat, не валидируют идею перед разработкой
- **Хотят:** Простое общение без жаргона, чёткий scope, рабочий продукт для инвесторов
- **Как продавать:** На языке бизнеса, не технологий

### 5.2 Funded стартапы (pre-seed / seed)

- **Бюджет:** $10K-50K
- **Профиль:** Привлекли $100K-$3.5M, runway ограничен
- **Боли:** Каждая неделя = сжигание runway, найм долгий и дорогой ($150K-200K/год)
- **Хотят:** Быстрый результат, production-quality, bridge до найма internal team
- **Как продавать:** "Заменяю первые 3 месяца найма"

### 5.3 Соло-фаундеры, которые застряли (растущий сегмент)

- **Бюджет:** $2K-8K
- **Профиль:** Начали строить сами в Cursor/Bolt/Lovable, упёрлись в стену
- **Боли:** Приложение не деплоится, AI-фиксы создают новые баги, сложные фичи не получаются
- **Хотят:** Починить и довести до production, не переписывать с нуля
- **Как продавать:** "Застряли с AI-кодом? Починю и задеплою за 1-2 недели"
- **Конкуренты:** [VibeCodeRescue](https://vibecoderescue.com/), [VibeRescue.dev](https://vibe-rescue.dev/), [Solveita](https://solveita.com/vibe-code-rescue), [Ronas IT](https://ronasit.com/services/vibe-coding-rescue/)

### 5.4 SMB, хотящие AI-фичи

- **Бюджет:** $5K-25K
- **Профиль:** Компании 10-500 сотрудников, оборот $500K-50M, есть продукт, нет AI-экспертизы
- **Боли:** Не понимают что реально в AI, боятся отставания от конкурентов
- **Хотят:** Конкретный ROI, интеграция в существующий продукт
- **Как продавать:** "AI-чатбот сократит нагрузку на поддержку на 40%"

### 5.5 Корпоративные innovation teams

- **Бюджет:** $15K-75K
- **Профиль:** Innovation/R&D отделы крупных компаний
- **Хотят:** Быстрый proof-of-concept для внутреннего питча
- **Как продавать:** Через LinkedIn, личный нетворк. Самый сложный сегмент для solo.

## 7. Тренды и риски

### Тренды
- **Ценовая компрессия** — что стоило $50K, теперь стоит $5-10K. Риск гонки на дно
- **Все обещают "MVP за 1-2 недели"** — нужна уникальность в позиционировании
- **Подписочная модель растёт** — вслед за DesignJoy, dev-as-a-subscription набирает обороты
- **Rescue-сервисы** — целый новый рынок "починки" AI-кода, уже 5+ компаний
- **Дизайн = дифференциатор** — AI-код быстрый но generic, хороший дизайн = premium pricing
- **Post-MVP retention** — лучшие бизнесы конвертируют разовые проекты в ретейнеры

### Ограничения solo AI-разработки
- **Не подходит для:** fintech/banking, healthcare (HIPAA), high-load (10K+ concurrent), enterprise с complex access control, legacy integration
- **~45% AI-кода содержит security flaws** ([Veracode 2025](https://www.veracode.com/))
- **AI-код потребляет 3-4x больше серверных ресурсов**
- **Один point of failure** — болезнь = проект стоит
- **Архитектурная inconsistency** — разные промпты дают разные паттерны

### Оптимальное позиционирование
> "Production-ready MVP за 2-4 недели и $5K-15K для валидации идеи на реальных пользователях. В 5-10x дешевле и быстрее классической разработки. Если идея взлетит — масштабируем."

### 8.3 Upwork-модель для вайб-кодера ($750/день)

5. **Ценообразование:** Начальная ставка $125-250, но после Loom-демо → переход к $2,500-5,000 за MVP
6. **Ретейнер:** $50-100/мес за хостинг + платные feature requests
7. **Масштаб:** 10-20 рекуррентных клиентов = предсказуемый income

## 10. Формат работы: Async + Crypto

### 10.1 Async-only коммуникация (без созвонов)

**Эталонная модель (DesignJoy / Poket Dev):**

| Элемент | Реализация |
|---------|-----------|
| Приём задач | Trello / Linear / Notion — клиент пишет задачу в свою доску |
| Обсуждение | Комментарии в задачах, Telegram/Slack-чат |
| Демо результата | Loom-видео (3-5 мин запись экрана с голосом) |
| Уточнения | Голосовые в Telegram, аудио-заметки |
| Документация | Notion/Google Docs — всё зафиксировано текстом |

**Преимущества async:**
- Клиент потребляет информацию когда удобно, без привязки к таймзоне
- Ничего не забывается — всё записано
- Экономия 30-40% времени (которое в sync уходит на митинги)
- Работает через любые часовые пояса

**Как реализовано у Poket Dev ($5,995/мес):**
1. Клиент подписывается → получает личную Trello-доску
2. Пишет задачи (unlimited requests)
3. Активирует одну задачу → разработчик начинает
4. Готовый код через GitHub за ~72ч
5. **Ноль созвонов. Вообще.**

- Без банковских комиссий ($0.01-2 вместо $25-50 wire fee)

- Стабильно (USDT/USDC = $1, без волатильности)
