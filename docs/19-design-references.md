## 6. Тренды дизайна 2025-2026

### Общие тренды портфолио/лендингов:
- **Тёмная тема** — доминирует, особенно в dev/tech нише
- **Bold typography** — крупные шрифты, акцентные заголовки
- **Микроанимации** — scroll-triggered, hover effects, но не перегруженные
- **Monochrome + 1 accent color** — чёрный/белый + один яркий цвет (зелёный, фиолетовый, оранжевый)
- **Kinetic text** — анимированный текст в hero-секции
- **Grid-based layouts** — структурированные сетки для кейсов/проектов
- **Минимализм** — меньше элементов, больше whitespace

### Конверсионные паттерны:
- **Первые 600px решают** — headline + value prop + CTA видны без скролла
- **Один CTA = +62% конверсии** — не распыляй внимание
- **10-20 секунд** — столько пользователь даёт странице до ухода
- **Social proof рядом с CTA** — отзывы/цифры возле кнопки

## 7. Выбранный стиль: Linear

> Решение: берём за основу дизайн-систему [linear.app](https://linear.app/) — тёмная тема, минимализм, glow-эффекты, продуктовый feel.

### 7.1 Дизайн-система Linear — конкретные детали

#### Цвета

| Токен | Значение | Использование |
|-------|----------|--------------|
| Background | ~1-10% lightness, НЕ чистый #000 | Основной фон |
| Mercury White | `#F4F5F8` (rgb 244,245,248) | Текст на тёмном фоне |
| Nordic Gray | `#222326` (rgb 35,35,38) | Карточки, панели |
| Brand Blue | Desaturated blue (subtle) | Акценты, ссылки |
| Градиенты | Сложные, многослойные, часто с glow | Hero-секция, акценты |

**Ключевое:** НЕ использовать чистый чёрный (#000000) — всегда 1-10% lightness для глубины.

#### Типографика

| Элемент | Шрифт | Примечание |
|---------|-------|-----------|
| Заголовки | **Inter Display** | Более выразительный, bold |
| Основной текст | **Inter** | Чистый, читаемый |
| Код | Monospace (var) | Для технических блоков |

**Альтернативы Inter:** Geist (Vercel), Satoshi, General Sans — похожий feel.

#### Визуальные эффекты

| Эффект | Описание | Где применять |
|--------|----------|--------------|
| **Glassmorphism** | Полупрозрачные карточки с blur | Pricing cards, feature cards |
| **Gradient glow** | Мягкое свечение за элементами | Hero headline, CTA buttons |
| **Noise overlay** | Лёгкая текстура поверх градиентов | Background, cards |
| **Subtle borders** | Тонкие `1px` borders с opacity | Карточки, разделители |
| **Scroll animations** | Fade-in при скролле, parallax | Секции при появлении |

#### Принципы layout

- **Линейный скролл** — глаза идут сверху вниз, без зигзагов
- **Последовательная компоновка** — одно направление чтения
- **Минимум CTA на экран** — 1 основной call-to-action
- **High-contrast текст** — тёмнее в light mode, светлее в dark mode
- **Много воздуха** — generous padding/margin

### 7.2 Конкретные CSS-приёмы для Linear-стиля

```css
/* Фон — НЕ чистый чёрный */
background: #0a0a0b;

/* Градиентный glow за hero */
background: radial-gradient(
  ellipse 80% 50% at 50% -20%,
  rgba(93, 93, 255, 0.15),
  transparent
);

/* Glassmorphism карточки */
background: rgba(255, 255, 255, 0.03);
backdrop-filter: blur(12px);
border: 1px solid rgba(255, 255, 255, 0.06);
border-radius: 12px;

/* Noise overlay */
background-image: url("data:image/svg+xml,..."); /* noise texture */
opacity: 0.03;

/* Gradient text (для headline) */
background: linear-gradient(to bottom right, #fff, #999);
-webkit-background-clip: text;
-webkit-text-fill-color: transparent;

/* Subtle glow на hover */
box-shadow: 0 0 20px rgba(93, 93, 255, 0.1);
transition: box-shadow 0.3s ease;

/* CTA button glow */
background: linear-gradient(135deg, #5d5dff, #7c3aed);
box-shadow: 0 0 30px rgba(93, 93, 255, 0.3);
```

### 7.4 Предлагаемая структура лендинга

```
1. Hero
   - Gradient glow фон
   - H1: bold, gradient text (Inter Display)
   - Подзаголовок: серый (#999), 1-2 строки
   - CTA: "Написать в Telegram" (glow button)
   - Под CTA: "Оплата в USDT • Без созвонов • 2-4 недели"

2. Social Proof Bar
   - Логотипы инструментов: Cursor, Claude, v0, Next.js
   - Или: "44% SaaS управляются соло-фаундерами"

3. Problem
   - 3-4 карточки (glassmorphism) с болями:
     "Агентства берут $50K+", "Фрилансеры пропадают",
     "Cursor-проект не деплоится", "CTO стоит $200K/год"

4. Solution → "Как это работает"
   - 4 шага, вертикальный timeline:
     Описание идеи → PRD + прототип → Разработка → Production
   - Иконки + короткие описания

5. Pricing
   - 3 карточки (glassmorphism):
     Starter $5K | Pro $8K | Подписка $5K/мес
   - Highlight на Pro (рекомендуемый)
   - Badge: "Оплата в USDT/USDC"

6. Comparison Table
   - 3 колонки: "Я" vs "Агентство" vs "Фрилансер"
   - Строки: Цена, Сроки, Коммуникация, Код

7. Portfolio / Case Studies
   - 2-4 карточки с mockup-скриншотами
   - Название + описание + результат
   - (Пока placeholder, потом заменим реальными)

8. FAQ
   - Accordion, 5-7 вопросов
   - Dark cards style

9. Final CTA
   - "Опишите идею — я оценю за 24 часа"
   - Telegram button (glow)
   - "USDT • Async • 2-4 недели"
```
